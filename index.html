Introduction

Cloud Computing Defenition: "the practice of using a network of remote servers hosted on the Internet to store, manage, and process data, rather than a local server or a personal computer. "

In scope of this presentation:
  • Google Docs

Out of scope:
  • Google Drive

Google Docs

Why is the Cloud Important to Business and Technology?

  • Ease of access
  • Easier to collaborate ideas and information with others
  • A tool we will have to use in the workforce (especially if IT related)
  • Cloud usage is increasing
Pro's and Con's of the Cloud

Pro's
  • Ease of access
  • Very inexpensive
  • No more software expenses
  • Data centralization
  • Data recovery

Con's
  • Need to have access to the internet
  • Not great qualitity
  • No physcial copy of your data
In Conclusion

  • Cloud Computing is a way to store, process and manage data using remote servers hosted on the internet
  • Alot of the applications we use on a day to day basis are via the cloud
  • Provides ease of access
  • Cloud usage is increasing
